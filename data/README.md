# AI Semantic Search using Endee Vector Database

## Project Description
This project implements a Semantic Search and Retrieval Augmented Generation (RAG) system using Endee as a vector database.

## Objectives
- Use Endee for vector storage
- Perform semantic search on documents
- Demonstrate a practical AI/ML use case

## Technologies Used
- Python
- Endee Vector Database
- Sentence Transformers

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Store documents  
   python ingest.py

3. Perform semantic search  
   python search.py

## Use Case
This project can be used for:
- AI-powered document search
- Knowledge base systems
- RAG-based chatbots
